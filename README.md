# Tool Researcher

Tool Researcher is a GitHub template for researching a tool, its features, and the competitive landscape around it. Repositories created from this template are meant to help a user determine whether a tool makes sense to invest in.

The template is built around an AI-guided research workflow. The user is first taken through a set of clarifying questions so the evaluation is grounded in the right context, goals, and constraints. Based on those answers, the AI engine performs research from the necessary perspectives, helping build a complete picture of the tool's strengths, gaps, risks, and market position. The results are then presented in an online-friendly format using Markdown documents, Mermaid diagrams, tables that are easy to navigate, and a one-pager that summarizes the key findings.

## What This Template Supports

- Researching a tool in a structured way
- Understanding product features and differentiators
- Analyzing the competitive landscape
- Evaluating whether the tool is worth further investment
- Guiding the user through clarifying questions before research begins
- Using AI to tailor the research based on the user's answers
- Presenting results in navigable Markdown, diagrams, and tables
- Providing a one-pager for a quick summary of the research

## How It Works

1. Create a repository from this template.
2. Answer the clarifying questions provided by the AI engine.
3. Let the AI engine research the tool from the relevant perspectives.
4. Review the findings in Markdown pages, Mermaid diagrams, tables, and a one-pager.
5. Use the research to decide whether the tool is a good fit and worth investing in.
